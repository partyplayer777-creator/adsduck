# 애즈덕 AI강의레터 구현 체크리스트

## 개요

AI 교육 정보를 게시하는 유료 콘텐츠 서비스이다. 유저는 포인트를 사용해 게시글을 1회 읽거나, 특정 게시글을 구독하거나, 전체 정기구독권을 구매할 수 있다.

기본 정책:

- 정기구독권: 1개월 50,000P, 3개월 120,000P, 6개월 200,000P
- 게시글 최초 읽기 포인트: 500P
- 게시글 최초 구독 포인트: 500P
- 게시글 구독 발생 시 해당 게시글의 읽기 포인트와 구독 포인트가 각각 100P 상승
- 유저는 게시글을 최대 3회까지 읽을 수 있음
- 3회 안에 게시글 구독을 하지 않으면 해당 유저에게 해당 게시글은 잠김
- 잠긴 게시글은 현재 게시글 구독 포인트의 5배를 지불해 구독하면 다시 열람 가능
- 게시글 구독자는 해당 게시글을 이후 무료 열람
- 정기구독권 보유자는 모든 게시글 열람 가능
- 일반 포인트 결제가 발생하면 글쓴이는 결제 포인트의 70%를 수익으로 받음
- 정기구독권 보유자가 게시글을 읽는 경우 글쓴이에게 게시글별 70% 수익은 지급하지 않음
